# Работа с Github

Гитхаб инструменты используешь когда пользователь ЯВНО просит поработать с Github.

Когда просят поработать с Github ты используешь Github MPC и его инструменты.
Например `get_issue` для чтения задач.

Для добавления комментария к задаче используй `add_issue_comment`

Для создания коммита используй `get_commit`
