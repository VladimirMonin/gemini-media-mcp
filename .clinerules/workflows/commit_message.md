# Работа с коммитами

Мы используем для коммитов инструмент MCP - Github и его возможности

Формат коммит-сообщения:

```
<префикс типа>: <краткое описание>
- <тезис 1 выполненной задачи>
- <тезис 2 выполненной задачи>
#номер_задачи
```

Префиксы типов:

- feat: новая функциональность
- fix: исправление ошибки
- docs: изменения в документации
- refactor: изменения в коде без изменения функциональности
